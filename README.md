# VCBOT
Fully working VC (user)Bot, based on py-tgcalls and py-tgcalls-wrapper with minimal [features](#TODO).   


## Deploying
* To heroku:   
[![Deploy](https://www.herokucdn.com/deploy/button.svg)](http://heroku.com/deploy?template=https://github.com/RayOfficial/MusicOfficial)   

* Local machine/VPS:   
`git clone https://github.com/RayOfficial/MusicOfficial`   
`pip install -r requirements.txt`   
`apt-get install ffmpeg`   
`touch .env && nano .env`, fill in the vars as in [.env.sample](./.env.sample)   
`python bot.py`   

## SESSION
Either run [sessiongen.py](./sessiongen.py) locally or run it on [repl.it by clicking here.](https://replit.com/@TeamUltroid/UltroidStringSession)
## Commands:   
- `!on` - Check if the (user)bot is online.   
- `!help` - Help message.   
- `!stream` - Either give a youtube URL or reply to a telegram file to play it.   
- `!pause` - Pause the stream.   
- `!resume` - Yes, resume.
